# 🎂 Birthday Surprise Website 🎉

This is a special **Birthday Celebration Website** created using **Next.js, Tailwind CSS, Framer Motion, and Lucide Icons**.
It’s designed as a heartfelt digital surprise — because sometimes, when you can’t be there in person, your code can express everything. 💖

---

## 🧠 Project Idea

> **POV:** It’s her birthday, but you can’t meet — so you build something unforgettable.

This project is not just a website, it’s an emotion crafted with code.

The website includes:

* ⏳ A live countdown timer
* 🎈 Personalized birthday wishes
* ✨ Smooth animations using Framer Motion
* 💌 Aesthetic UI with meaningful design

It captures a journey — from writing code in VS Code to delivering a beautiful surprise in the browser.
